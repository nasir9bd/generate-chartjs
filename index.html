1. First Attach Chartjs library to script area

<script src="//cdnjs.cloudflare.com/ajax/libs/Chart.js/2.1.4/Chart.min.js"></script>

2. Add canvas tag to html area

<canvas id="myChart" width="400" height="400"></canvas>

3. Copy the javascript code to your script js file

var ctx = document.getElementById('myChart').getContext('2d'); var data = {{chartData.data | json}}; var options = {{chartData.options | json}}; var myChart = new Chart(ctx, { type: '{{chartData.type}}', data: data, options: options });
